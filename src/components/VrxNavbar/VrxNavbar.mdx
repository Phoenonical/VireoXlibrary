# VireoxNavbar

VireoxNavbar is a customizable component for using a navigation menu.

To use it, import the component and use it as follows:
```js
import {VrxNavbar} from 'vireoxlibrary';

<VrxNavbar buttons="[ ... ]"  >
    <template v-slot:leftComponent>
    </template>
</VrxNavbar>
```
The ```buttons``` prop must be an array ```[]``` of objects of type ```NavbarButton```. Below are the interfaces to use when configuring the navbar.

```NavbarButton```
The main button of the navbar.

```js
type NavbarButton = {
    text: string;                               // Button text
    icon: IconLibraryType | string;             // [optional] Button icon or SVG path
    component: ComponentInterface;              // [optional] Component to be inserted into the button
    children: FirstLayerButton[];               // [optional] Sub-buttons of the button
}

```

```NavbarFirstLayerButton```
A child sub-button of a main button. It can contain another level of sub-buttons.

```js
type NavbarSecondLayerButton = {
    text: string;                               // Button text
    description: string;                        // [optional] Button description
    icon: IconLibraryType | string;             // [optional] Button icon or SVG path
    component: ComponentInterface;              // [optional] Component to be inserted into the button
    children: SecondLayerButton[];              // [optional] Sub-buttons of the button
}
```

```NavbarSecondLayerButton```
A child sub-button of a FirstLayerButton. The last layer of buttons.

```js
type NavbarSecondLayerButton = {
    text: string;                               // Button text
    component: ComponentInterface;              // [optional] Component to be inserted into the button
}
```

```ComponentInterface```
A component (primarily intended for routing) to be inserted into a navbar button. To display the button text and description, the passed component must have a slot.
It is recommended to use ```<NuxtLink> </NuxtLink>``` or ```<router-link to="/"> </router-link>``` for routing.
```js
type ComponentInterface = {
    component: Component;                       // Component object to be inserted into the button
    props: any;                                 // [optional] Object with props to be passed to the component
}
```
