# VireoxButtonsGroup

VireoxButtonsGroup è un componente che permette di ragruppare più bottoni.

Per utilizzarlo, importare il componente e utilizzarlo come segue:
```js
import { VrxButtonsGroup } from 'vireoxlibrary';

<VrxButtonsGroup
    buttons="['Option 1', 'Option 2', 'Option 3']"
    actions="[() => alert('Option 1'), () => alert('Option 2'), () => alert('Option 3')]"
/>
```

Per dettagli sulle API e le configurazioni leggere la [documentazione](./?path=/docs/vrxbuttonsgroup--docs)
