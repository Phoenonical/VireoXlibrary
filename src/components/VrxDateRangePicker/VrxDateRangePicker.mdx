# VireoxDateRangePicker

VireoxDateRangePicker is a "calendar" component for date range selection.
The component exclusively utilizes the Date class in Javascript for date handling.

To use it, import it as follows:
```js
import {VrxDateRangePicker} from 'vireoxlibrary';

<VrxDateRangePicker type="date" />
```
