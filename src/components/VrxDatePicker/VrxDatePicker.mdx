# VireoxDatePicker

VireoxDatePicker is a "calendar" component for date selection.
The component exclusively utilizes the Date class in Javascript for date handling.

To use it, import it as follows:
```js
import {VrxDatePicker} from 'vireoxlibrary';

<VrxDatePicker type="date" />
```
