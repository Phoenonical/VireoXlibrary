# VireoxToggle

VireoxToggle is the alternative component to the traditional checkboxes.

To use it, import it as follows:
```js
import {VrxToggle} from 'vireoxlibrary';

const value = ref(false);
<VrxToggle v-model="value" />
```

For API's details and configuration please read the [documentation](./?path=/docs/vrxtoggle--docs)
