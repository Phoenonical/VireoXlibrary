# VireoxSelect

VireoxSelect is the basic input component for single and multiple selection. The component utilizes 100% of the available width.

To use it, import it as follows:
```js
import {VrxSelect} from 'vireoxlibrary';

const value = ref([])
const data : SelectItemInterface[] = [
    {value: '1', label: 'Opzione 1'},
    {value: '2', label: 'Opzione 2'},
    {value: '3', label: 'Opzione 3'},
]
<VrxSelect v-model="value" :list-data="data"/>
```

For API's details and configuration please read the [documentation](./?path=/docs/vrxselect--docs)
